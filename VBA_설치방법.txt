재고표 VBA 매크로 설치 방법
====================================

1단계: 본두리편의점 재고실사.xlsx 파일 열기

2단계: 매크로 활성화
   - Alt + F11 키 누르기 (VBA 편집기 열림)

3단계: 모듈 삽입
   - 왼쪽 프로젝트 창에서 "VBAProject (본두리편의점 재고실사.xlsx)" 우클릭
   - 삽입 > 모듈 클릭

4단계: VBA 코드 복사
   - 재고표_VBA코드.bas 파일 내용 전체 복사
   - VBA 편집기 오른쪽 코드 창에 붙여넣기

5단계: 저장
   - 파일 > 다른 이름으로 저장
   - 파일 형식: Excel 매크로 사용 통합 문서 (*.xlsm) 선택
   - 파일명: 본두리편의점 재고실사.xlsm
   - 저장 클릭

6단계: 버튼 만들기 (선택사항)
   - VBA 편집기 닫기
   - 개발 도구 탭 > 삽입 > 단추(양식 컨트롤) 클릭
   - 시트에 버튼 그리기
   - "매크로 지정" 창에서 "재고표_생성" 선택
   - 확인 클릭
   - 버튼 이름을 "[재고표 생성]"으로 변경


사용 방법
====================================

방법 1: 버튼 클릭
   - [재고표 생성] 버튼 클릭
   - 자동으로 "재고표_출력" 시트 생성

방법 2: 매크로 직접 실행
   - Alt + F8 키
   - "재고표_생성" 선택
   - 실행 클릭


결과
====================================

- "재고표_출력" 시트 자동 생성
- 정리표 데이터를 재고실사양식표 형식으로 변환
- 제품코드, 제품명, 아주/날짜, 일반/날짜, 잔량/날짜
- 형식: 16(11/25) - 숫자(월/일)
- 수량 부분 12pt, 굵게 강조
- 바로 인쇄 가능
