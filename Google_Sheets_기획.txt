음료회사 주문/라벨/재고 자동화 시스템
Google Sheets 버전

========================================
무엇을 하는 프로그램인가요?
========================================

SAP에서 받은 주문 데이터를 Google Sheets에 붙여넣으면
자동으로 업체별 주문서와 팔레트 라벨을 PDF로 만들어줍니다.


========================================
어떤 기능이 있나요?
========================================

1. 데이터 자동 정리
   - 마이너스 수량(반품) 자동 삭제
   - 업체명을 코드로 자동 변환
     예) BGF화성 -> 중부, GS발안 -> 209

2. 업체별 주문서 PDF 자동 생성
   - 업체마다 PDF 파일 1개씩 생성
   - 그 업체에 보낼 제품과 수량이 표로 정리됨
   - 예) 중부_주문서_20251217.pdf

3. 팔레트 라벨 PDF 자동 생성
   - 업체마다 PDF 파일 1개씩 생성 (여러 페이지)
   - 각 페이지는 제품 1개의 라벨 (A4 용지 1장)
   - 큰 글씨로 제품명, 수량, 업체코드 표시
   - 예) 중부_라벨_20251217.pdf (13페이지)

4. 재고표 PDF 자동 생성
   - 유통기한 빠른 순서로 자동 정렬
   - 선입선출용 재고표 PDF로 출력


========================================
어떻게 사용하나요?
========================================

1단계: SAP 데이터 붙여넣기
   - SAP에서 주문 데이터를 다운로드
   - Google Sheets 열기
   - 원본_주문데이터 시트에 붙여넣기

2단계: 데이터 정리 (10초)
   - 메뉴 클릭: 자동화 > 데이터 처리하기
   - 마이너스 수량 삭제, 업체명 변환 자동 완료

3단계: 주문서 만들기 (3분)
   - 메뉴 클릭: 자동화 > 주문서 PDF 생성
   - 잠시 대기
   - Google Drive에 PDF 파일 50개 자동 저장

4단계: 라벨 만들기 (3분)
   - 메뉴 클릭: 자동화 > 라벨 PDF 생성
   - 잠시 대기
   - Google Drive에 PDF 파일 50개 자동 저장

5단계: 인쇄
   - Google Drive > 자동화_출력 폴더 열기
   - 필요한 PDF 다운로드해서 인쇄

========================================
시간이 얼마나 걸리나요?
========================================

기존 수동 작업: 2-3시간

자동화 후:
   - 데이터 붙여넣기: 2분
   - 데이터 정리: 10초 (자동)
   - 주문서 생성: 3분 (자동)
   - 라벨 생성: 3분 (자동)
   - PDF 다운로드 및 인쇄: 2분

총 10분 소요
